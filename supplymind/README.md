# SupplyMind — Next.js App (Legacy)

This directory contains the older Next.js-based app used for prototyping and landing pages. The main app lives in `frontend/` (React + Vite + TypeScript).

## Getting Started

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).

## Auth

Uses Clerk (`@clerk/nextjs`) for authentication in this sub-project.
